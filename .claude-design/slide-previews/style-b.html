Skills vs SubAgents

// Skills

  • Depth — deep domain expertise in one area
  • Reusable — loaded on demand, session after session
  • Persistent knowledge — survives context resets
  • Runs in-context — enriches the main agent

// SubAgents

  • Breadth — parallel execution across many files
  • One-shot — spun up, does work, reports back
  • Parallel execution — independent workers in parallel
  • Isolated context — protects main agent's window
